Meu aprendizado

Durante o módulo, denominado Fundamentos, tive contato com vários conteúdos. Nesse módulo foi possível aprender tópicos básicos de comandos linux, tais como criar e remover diretórios. Aprendemos também como utilizar a ferramenta GitHub usando os comandos git status, git add, git commit e git push. Outro tema muito importante foi a introdução ao HTML e CSS. Aprendemos vários conceitos básicos e como utilizar as principais Tags do HTML. Fomos iniciados em pontos importantes como o HTML semântico, bem como estilizar a nossa página HTML por meio do CSS Style.
