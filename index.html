Artigo

Minha jornada na Trybe mal começou e já aprendi muito. Não sabia nada de Git e muito menos de HTML. E estou aqui, em 3 semanas construindo página do zero para meu primeiro projeto! Claro que ainda não sei o sufiente para fazer um página bonitona, mas para quem começou a aprender do zero já é incrível...Eu estou muito feliz! Ansioso para aprender mais, o suficiente para começar a fazer páginas cada vez mais bonitas!
